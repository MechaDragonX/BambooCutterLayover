Welcome to Bamboo Cutter Layover!

Bamboo Cutter Layover a basic website that takes a URL from a manga reader, and creates a redirect link to guya.moe, a robust comic book reader that focuses on Kaguya-sama: Love is War, using its proxy feature.

What does this mean!?

It just means that you can read your favorite manga using the reader that guya.moe has, which I personally think is the best comic book reader due to a number of cool features.
